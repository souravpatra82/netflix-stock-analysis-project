<h2 id="overview">📌 Overview</h2>

<p>
This project analyzes Netflix stock market performance from 2018 to 2022 using 
Excel, Power BI, Python, MySQL, and CSV datasets. The project focuses on stock trends,
market sentiment, trading volume, and KPI performance analysis.
</p>

<hr>

<h2 id="objectives">📊 Project Objectives</h2>

<ul>
  <li>Analyze Netflix stock price trends</li>
  <li>Measure trading volume performance</li>
  <li>Track stock return percentage</li>
  <li>Visualize market sentiment trends</li>
  <li>Create interactive business dashboards</li>
</ul>
